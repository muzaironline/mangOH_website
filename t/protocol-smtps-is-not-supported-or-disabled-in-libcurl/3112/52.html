dfrey January 14, 2020, 4:11pm 48

Instead of doing make image_bin, you can run USE_DOCKER=1 make image_bin and that will cause the Makefile to use a docker image to perform the build instead of your native Linux environment. Basically, as long as you have docker installed, this should work without any other dependencies. At that point it doesn’t matter what version of Ubuntu you’re running or if you’re running Ubuntu at all since the docker container provides the appropriate version of Ubuntu and the necessary packages.
