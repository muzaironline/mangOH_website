jyijyi December 25, 2020, 12:32pm 5

If NMEA is added to the definition file as a device, it cannot be accessed by the sandbox app:

requires:
{
device:
{
/dev/nmea /dev/
}
}

But, if adding it as a file, it works:

requires:
{
file:
{
/dev/nmea /dev/
}
}
