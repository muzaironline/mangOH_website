OK, I could finally try this out. Sorry it took me so long to come back to you on this, @jdots83 . I’ve been quite busy the last few days and didn’t get the time to do it.

This is definitely the RIGHT answer, not sure if this should be the final solution. We had some issues regarding AV usage, specifically with the “administrative” part. As @jdots83 stated, you need another user and a “new” company in order to use this solution. Keep in mind, this requires another “company” email address. Also, you need to remove the mangOH board from the current AV account, which seems obvious. Sadly, both this tasks are easier said than done, at least from the AV web GUI.

AV user and mangOH device removal, is not straightforward. This is actually why we had to get in touch with SiWi in the first place. Up to this moment, we couldn’t delete all our users from the current company, so we needed SiWi to do this for us. In order to “delete” the mangOH device, you have to remove the information from the mangOH paired with the Sierra SIM card in AV, i.e. remove all info in “Gateway” and click on “Save”:

image.png1235×477 25.7 KB

After this, you can create the mangOH device in the new AV account. This seems like an unnecessary workaround for a task that SHOULD be easy. Perhaps there’s an easier way to do this… We’ve been talking to the SiWi guys (thanks D. Lahay for all the assistance!) and they’re currently working hard on getting a “seamless experience” using mangOH with AV.
