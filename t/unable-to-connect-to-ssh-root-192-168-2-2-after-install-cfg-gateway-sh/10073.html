In cfg_gateway.sh, you need to add "iptables -I INPUT -j ACCEPT＂ to shutdown the firewall after “iptables --flush”


iptables --flush
if [ "$?" -ne 0 ]; then
    echo "Flush iptables failed"
    exit 1
fi
iptables -I INPUT -j ACCEPT
